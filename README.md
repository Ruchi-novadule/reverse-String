🔁 Reverse String in Python (Using Slicing)

This is a simple Python program that reverses a string using Python’s slicing method ([::-1]).
The program takes user input and prints the reversed string.

📌 Features

Takes input from the user

Uses Python slicing technique

Short and efficient implementation

Beginner-friendly

Clean and readable code

💻 Technology Used

Python 3

🚀 How It Works

The program uses Python slicing:

[::-1]

: → start to end of string

-1 → step backwards

This reverses the string in one line

📝 Code
user_input = input("Enter a string: ")
reversed_string = user_input[::-1]
print("Reversed string is:", reversed_string)
▶️ Example

Input:

hello

Output:

Reversed string is: olleh
